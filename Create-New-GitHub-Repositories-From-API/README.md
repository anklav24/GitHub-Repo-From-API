# Delete-GitHub-Repositories-From-API

### Для чего предназначен проект:

Скрипт предназначен для создания репозитория на GitHub локально с компьютера не заходя в Web интерфейс.
При этом будет создана папка со служебными файлами типа .gitignore, readme.md и т.д.

### Инструкции по конфигурации и установке:

Запустить скрипт, следовать подсказкам.

Так же можно добавить в код свои данные (Имя профиля, токен API от GitHub и т.д.)

### Примеры использования:

Ускоряет создание новых репозиториев.
