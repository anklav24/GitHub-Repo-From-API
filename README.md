# GitHub-Repo-From-API
### Для чего предназначен проект:

Скрипты предназначеные для создания/удаления репозиториев на GitHub локально с компьютера не заходя в Web интерфейс.

### Инструкции по конфигурации и установке:

Запустить скрипт, следовать подсказкам.

Так же можно добавить в код свои данные (Имя профиля, токен API от GitHub и т.д.)

### Примеры использования:

Ускоряет удаление ненужных репозиториев.
